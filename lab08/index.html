Form Elements

The Select Element

The Select Element is used to define a drop down list. The first item in the drop down list will always be automatically selected. If a user wants to change that, they will need to add the "Selected" attribute next to the option value.

The Label Element

The Label Element is used to define and label something for the user. It is useful for the reader, because they will read it and know what type of input the program is asking the user for.

The TextArea Element

The TextArea Element defined a multi line input field for the user.

The Button Element

The Button Element defines a clickable button for the user.

The Output Element

The Output Element would display the result of a calculation performed in the coding.
