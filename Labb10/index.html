XML

XML stands for Extensible Markup Language. It was designed to store and trasnport data.

How is XML used?

XML lets you define and store data as well. XML supports the information exchanged between websites and databases.

What is JavaScript?

JavaScript is a programming language that allows you to implement many different features to web pages. It is used for web pages when there are things such as interactive maps, 2D or 3D graphics and much more.

How is JavaScript related to HTML and CSS?

HTML is used to structure content on web pages and CSS is used to style the content and add color and much more. JavaScript is used to add user interactive things to the web pages. JavaScript can have HTML and CSS change the content and appearance of a web page based on user activity..
